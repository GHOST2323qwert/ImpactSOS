# 🚨 ImpactSOS

ImpactSOS is a real Android emergency application that detects dangerous situations and allows the user to trigger an SOS alert.

---

## 📱 Features

- 🚨 SOS button
- 📞 Emergency trigger (ready for calls/SMS)
- 📍 Ready for location integration
- ⚡ Lightweight and fast
- ☁️ Ready for Firebase integration

---

## 🧠 How It Works

1. User opens the app  
2. Presses the SOS button  
3. App triggers emergency action (currently alert message)  

---

## 🛠️ Technologies

- Kotlin
- Android SDK
- Gradle
- GitHub Actions (APK build)

---

## ⚙️ Build APK (GitHub)

1. Go to **Actions**
2. Run **Build APK**
3. Download from **Artifacts**
